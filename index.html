HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Using lang="en" tells the browser the language for the webpage.
  • HTML attributes are always defined in teh beginning element, never the closing element.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Declarations contain the CSS property we want to apply and the value of the property
  • A margin in CSS is like the margin in word
  • The padding is confused with the margin, but the padding adds space aroudn teh content inside an element

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • typing code . will open VS Code
  • the feature/starter code is the name of the new branch
  • main is the main branch
  • the -b flag creates a new branch
  • the git command must precede every Git statement

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • You can use the console.log method to output a message to the web console by adding an argument
  • You use an assignment operator to assign the specific data that you want stored in the variable
